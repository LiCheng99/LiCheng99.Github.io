Hexo 搭建博客

1.0准备工作

  • 先准备要一会要用到的工具
    • Git
    • Node.js
    • Github
    • Vscode (代码编辑器 Vscode-Hbiuder任选其一)
    • Hbiuder (代码编辑器 Vscode-Hbiuder任选其一)

官方网址：Git https://git-scm.com/

这边已经准备好了(也可以自己去官方下载)：Git-2.40.0-64-bit.exe https://down.lccicc.com/down/hexo/Git-2.40.0-64-bit.exe

官方网址：Node.js https://nodejs.org/en

这边已经准备好了(也可以自己去官方下载)：node-v18.16.0-x64.msi https://down.lccicc.com/down/hexo/node-v18.16.0-x64.msi

官方网址：Github https://github.com

Github去注册一个账号就好 Github

官方网址：Vscode https://code.visualstudio.com/

官方网址：Hbiuder https://dcloud.io/hbuilderx.html

2.0工具安装

2.1 安装Git

  1. 安装就不讲了，傻瓜式安装就好
  2. 安装完成之后就需要配置
  1. 检查是否配置成功

2.2Git常用命令

2.3安装Node.js

  1. 安装就不讲了，傻瓜式安装就好

  2. 下载npm源，默认是从国外下载的，比较慢，那我们就改成淘宝镜像源，cmd窗口输入

2.4安装Hexo

  1. 随便找一个文件夹，桌面也可以，右键选择Git Bash Here粘贴

2.5检查安装

  1. 安装完成需要的工具，检查是否都安装成功了

3.0连接到Github

  1. 注册Github，官方网址放在上面了注册一下就好

  2. 新建一个仓库，右上角+号新建仓库

    ​

  3. 填写仓库信息

    1. 注意命名 Repository name 必须是 <you_name>.github.io <you_name> 是你的Github用户名

    2. 描述-Description-可填可不填

    3. 仓库权限必须是 Public 公共显示

    4. 最后-Description-创建储存库就好了

      我的显示为红色的是因为我已经有这个仓库了，所以不能命名为 LiCheng99.github.io 了

  4. 生成ssh公钥，这个公钥是你的电脑连接Github的重要工具

    1. 随便找一个文件夹，桌面也可以，右键选择Git Bash Here粘贴
    1. 接着就会生成一个名为 id_rsa.pub 的文件，路径是 C:\Users\<you_user>\.ssh 为你电脑的用户名，用记事本打开，ctrl+a 全选复制
    2. 回到Github右上角头像下拉选择框里面的 Settings

  5. 找到左侧导航栏的 SSH and GPG keys 右上角 New SSH key 添加ssh公钥

  6. 添加ssh公钥就好了

  7. 添加完成，测试连接，Git Bash Here 输入

4.0初始化Hexo

  1. 进入你博客的文件夹根目录，例： D:\hexo-blog 鼠标右键 Git Bash Here 执行
  1. 输入 npm i 安装相关依赖
  1. 最后输入 hexo s 启动项目
  1. 访问 http://localhost:4000/ 就看到已经在本地部署成功了

5.0本地部署到Github Pages

  1. 安装hexo-deployer-git

  1. 修改博客根目录下的 _config.yml 文件

    1. 往下翻，一般在最后面

    2. 将 repository: 后面跟着你Github仓库的地址，去Github仓库里面复制

    3. branch: 写 main 就好

    4. 以上都做好之后开始上传到Github Pages

    成功后访问你的Github仓库名就好啦，没成功就多试几次，挂梯子会好一点，因为是国外的

希望本文能够帮助读者更好地搭建出Hexo博客。
